# skill-manager

AgentBox's built-in skill manager CLI. List, check, install, and scaffold skills.

## CLI
- **Type:** custom
- **Binary:** `agentbox-skill`
- **Repo:** https://github.com/jackgladowsky/agentbox (src/skill.ts)

## Auth
- **Required:** no

## Depends On
- none

## Commands
- `agentbox-skill list`
- `agentbox-skill check`
- `agentbox-skill show <name>`
- `agentbox-skill install <name>`
- `agentbox-skill add <name>`
