# rex-skill

Rex's built-in skill manager CLI. List, check, install, and scaffold skills.

## CLI
- **Type:** custom
- **Binary:** `rex-skill`
- **Repo:** https://github.com/jackgladowsky/agentbox (src/rex-skill.ts)

## Auth
- **Required:** no

## Depends On
- none

## Commands
- `rex-skill list`
- `rex-skill check`
- `rex-skill show <name>`
- `rex-skill install <name>`
- `rex-skill add <name>`
